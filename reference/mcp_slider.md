# Create an MCP slider input

Generates a range slider element with MCP data attributes.

## Usage

``` r
mcp_slider(id, label, min, max, value = min, step = 1)
```

## Arguments

- id:

  Input ID

- label:

  Display label

- min:

  Minimum value

- max:

  Maximum value

- value:

  Initial value

- step:

  Step increment

## Value

An
[htmltools::tag](https://rstudio.github.io/htmltools/reference/builder.html)
object
