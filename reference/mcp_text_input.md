# Create an MCP text input

Generates a text input element with MCP data attributes.

## Usage

``` r
mcp_text_input(id, label, value = "", placeholder = NULL)
```

## Arguments

- id:

  Input ID

- label:

  Display label

- value:

  Initial value

- placeholder:

  Placeholder text

## Value

An
[htmltools::tag](https://rstudio.github.io/htmltools/reference/builder.html)
object
